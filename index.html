Robotic Arm and Car
Java Script Joystick


We created a robot called “EvoCar” which is a 3D Printed Arduino and Raspberry Pi controlled car with a 6 servo robotic arm on top. In order to control the car we had created a webpage with buttons, up, down, left, right, and stop. But, clicking the buttons repeatedly, was getting very tedious. To solve this problem we created a joystick.

This joystick allows us to set/increase/decrease to any speed within range for both the X and Y axes simultaneously. The webpage can be used to send a left speed and right speed for the car. The calculation for these speeds is adding/subtracting the X value to/from the Y value. The adding or subtracting is based on which way the X axis is moved, left or right of the center. (July 19, 2020)

Java Script Sliders


Another issue that came up with EvoCar was the means to controlled the arm and each of its servos. In order to do this, I created a webpage for sliders purely in JavaScript for each servo and its position.

Each slider ranges from 0 to 180. This webpage can be used to move each servo to a position within 0 to 180 degrees. (August 12, 2020)

Arduino
LED Mirror



I attempted to create an Arduino controlled system which was to take the live video feed from a camera and light up a 10x10 array of LEDs resembling the camera input. The first step was to pixelate the camera feed to 10x10 and convert to greyscale. I did this using Processing by using the camera input and then splitting the width by 10 and the height by 10 and then getting the average color for each square. Then, I printed out the greyscale value for each square from 0-100 in a 10x10 array.







The next step was to create the physical representation. I tested this out with 4 LEDs in a 2x2 array. and instead of splitting the camera input into 10x10 I split it into 2x2. Then, using the Arduino, I sent the greyscale value for each square to the respective LED. This was the result:

Once more and more LEDs are added, the array of LEDs will start looking more and more like the real input feed and start to look more like a "mirror" ( June 23, 2020 )

COVID-19 Safety
Hands-Free Hand Sanitizer Dispenser


One of the main concerns of COVID-19 or any virus in general is the spread through surfaces that many people touch. Everyone always says, “use hand sanitizer” to try to lower the chances for any spread. However, in the majority of places such as doctor’s offices, schools, and even homes, the only type of hand sanitizer dispenser are the ones where you must press the top of the bottle with your hand in order to squeeze out the liquid, which defeats the purpose of “sanitizing.” Although there are many hands free dispensers with motion sensors, they can be expensive and harder to buy multiple. I designed a 3D printed foot operated dispenser using Fusion 360, with the additional parts only being a PVC pipe and a string. (July 27, 2020)
